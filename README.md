## Overview
This project is a simple web-based library application that allows users to manage their personal collection of books in an interactive and intuitive way.
Users can add new books by providing relevant details, remove books from the collection, and update the reading status of each book. The application dynamically updates the interface, ensuring a smooth and responsive user experience.


## Live Demo

Visit the website: [Live Demo](https://aderitomarrime.github.io/library/)


## Features

- Minimal interface  
- Grid Layout 
- Add new books to the library
- Remove books from the library
- Toggle book read status
- Dynamic UI updates without page reload
- Simple and user-friendly interface
- Store and manage multiple books

## Installation

To run this project locally, follow these steps:

1. Clone the repository:

```sh
git clone https://github.com/aderitomarrime/library.git
```

2. Navigate to the project 
```sh
cd library
```
3. Open index.html in your browser


## How to use

1. Open the website in your browser
2. View the list of books already added by default
3. Interact with existing books:
    - Remove a book from the library
    - Toggle its read status (read / not read)
4. Add a new book:
    - Click on the "New Book" button
    - Fill in the required information in the form
    - Submit the form to add the book to the library
5. Manage your new books the same way:
    - Remove them
    - Toggle their read status
